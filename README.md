# 🎬 Movies App

A modern movie discovery application built with **React** and powered by the **TMDB API**.  
Browse trending movies, search for your favorite titles, and explore detailed movie information with a clean and responsive interface.

## ✨ Features

- 🔍 Search for movies
- 🎞️ Browse trending and popular movies
- 📄 Detailed movie information
- ⭐ Movie ratings and release dates
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design
- 🔗 Real-time data fetched from TMDB API

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS3
- TMDB API
